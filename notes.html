Client / server:

Client-Server is a model or an architecture that describes the relationship between the clients and the servers. Servers provide resources and services to the requests processed by the clients(users).

Git:

Git is a version control system which can track the work done by different programmers or the changes done in the work done by a single programmer too. It is used in a distributed system and can be effective to integrate data.

Github:

Github is web application remote server which hosts the services of Git, while it has many of its own features. It provides a platform for the users to publish their softwares and share their software development processes.


Lab 2: 2 HTML Knowledge:

  • Html tag:

    The <html> tag gives the info to web browser that this is an html document. Examples of <html> tags are <head>,<body>,<title> etc

  • Tags which dont have content present in them are called Empty tags. These tags do not have an end tag. Example <br> tag....used for a line break.

  • Attributes of an html tag gives more info about that tag(element). Example href attribute is used for reference of links...used in <a> tag.

  • Nested tags : The tags that are opened first must be closed at the last. Like First in Last Out.

  • id attribute is used to specify a unique value within the html document, whereas the class attribute is used to define the equal styles for elements with the same class name.
